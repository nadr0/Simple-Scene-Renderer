# Simple-Scene-Renderer

A ray tracer built with C++. A lot of the code comes from *__Ray Tracing From The Ground Up__* -  [[site]](http://www.raytracegroundup.com/)

Features
-----

+ Lighting
    * Direct
    * Point
    * Area
+ Materials
    * Matte
    * Phong
    * Reflective
+ Primitives 
    * Sphere
    * Rectangle
    * Plane
    * Triangle
+ Tracers
    * Basic (non whitted)
    * Whitted
    * Path tracing (Global Illumination)
